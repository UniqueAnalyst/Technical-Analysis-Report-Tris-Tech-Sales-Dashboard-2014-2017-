## 🎯 Objective of the Project

The dashboard provides a comprehensive analysis of Tris-Tech’s sales performance. The goal is to:

- Analyze sales trends
- Identify key revenue drivers
- Evaluate shipping preferences
- Track customer behavior
- Highlight regional performance

All of this supports informed, strategic decision-making for the business.

## 📌 Post-Analysis & Insights

### 🧠 Key Findings

1. **Customer Concentration**  
   Revenue is heavily concentrated among a few top customers.

2. **Shipping Preferences**  
   Standard Class dominates, showing price sensitivity.

3. **Regional Strength**  
   West region (40%) is the strongest performer.

4. **Product Mix**  
   Technology products account for 52% of sales.

5. **Order Value Distribution**  
   Most orders are in the $0–$100 range.

6. **Seasonal Trends**  
   Sales peak in March–April and October–November.

7. **Top Product**  
   Canon ImageCLASS MF8580Cdw Copier: $230,995.34 revenue.

8. **Top State**  
   California leads with $674,981.49 in revenue.
